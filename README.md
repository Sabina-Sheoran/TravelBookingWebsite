# K-Stay — Travel Booking UI/UX Design

A high-fidelity Figma prototype for a modern travel booking platform.  
Designed as part of an HCI course project (MCA, AI & ML).

---

## Overview

K-Stay is a travel booking web application prototype focused on delivering a seamless and intuitive user experience. The design covers the complete booking flow — from discovery to confirmation.

## Design File

| File | Description |
|---|---|
| `Booking.fig` | Full Figma prototype (55 MB) — stored via Git LFS |

> **Note:** To open the design, download `Booking.fig` and import it into [Figma](https://figma.com).

## Key Screens

- **Home** — Hero section with search bar and destination highlights  
- **Search Results** — Filtered listing of available stays  
- **Property Details** — Gallery, amenities, reviews, and pricing  
- **Booking Flow** — Date selection, guest info, and payment  
- **Confirmation** — Booking summary and receipt  

## Design Highlights

- Clean, minimal aesthetic with a calm color palette  
- Consistent component library and design tokens  
- Mobile-first responsive layouts  
- Micro-interaction annotations for developers  

## Tools Used

- **Figma** — UI design & prototyping  
- **HCI principles** — User research, wireframing, usability
